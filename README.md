📊 Dashboard
Real-time organization metrics (Total Employees, Present Today, On Leave, etc.)
Interactive attendance widget with live working hours timer
Leave balance donut chart with visual legend
Company announcements, upcoming birthdays, payslip preview
Task management with check/uncheck functionality


👤 My Profile
Complete employee profile with personal, employment & financial details
Emergency contact information
Edit profile capability (UI ready for backend)
⏰ Attendance Management
Today's attendance with Check-In / Check-Out toggle
Live working hours counter
Monthly calendar view with color-coded status (Present, Late, Absent, Leave, WFH)
Attendance history table with export option
Attendance correction request form


📅 Leave Management
Leave balance cards (Casual, Earned, Sick, Unpaid)
Apply for Leave form with validation
Leave type selection
Duration (Full Day / Half Day / Multiple Days)
Date range picker
Reason & emergency contact
Leave policy guidelines
Leave history with filter tabs (All / Approved / Pending / Rejected)
Cancel pending leave applications


💰 My Payslips
Salary summary (CTC, Gross, Net, Tax)
Payslip history with download & view options
Monthly payment status tracking
📈 Performance
Overall rating display
Goals & KPI tracker with progress
Quarterly performance review history
Manager feedback display


📁 Documents
Grid view of all employee documents
Offer Letter, Appointment Letter, Certificates, IDs, HR Policies
One-click document access (ready for PDF integration)


👥 Company Directory
Employee cards with avatars, roles & departments
Search & filter by department/location
Employee status indicators


📆 Calendar
Full month calendar view
Today's events list
Upcoming holidays
Birthday reminders


✅ Tasks
Add new tasks via modal
Task tabs: All / Pending / Completed / Overdue
Priority levels & due dates
Project/category tags


📢 Announcements
Detailed announcement cards with icons
Posted by, date & view count
Company news & policy updates

Account: Profile picture, name, email, phone, language, timezone
Notifications: Email, push, attendance reminders, leave alerts, payslip notifications
Security: Password change, 2FA toggle, login alerts, active sessions
Appearance: Dark mode, compact sidebar, font size

FAQ accordion with common questions
